Bom Dia!

A tag "h" serve para adicionar cabeçalhos.

A tag "title" serve para nomear o documento.

A tag "body" é literalmente o corpo do documento.

A tag "html" é a raiz do documento, junta todos os elementos.

A tag "head" é a cabeça do documento que compreende as suas informações.

A tag "p" serve para adicionar um novo parágrafo

A tag "u" serve para sublinhar a escrita.
